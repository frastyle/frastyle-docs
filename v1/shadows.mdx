## Bayangan

Perlengkapan tambahan berupa bayangan dapat ditambahkan pada sebuah elemen dengan menuliskan class `shadow` atau `shadow-[tipe]`. Perlengkapan bayangan ini, juga mendukung penggunaan [breakpoint responsif](./responsive) dengan format penulisan `shadow-[breakpoint]-[tipe]` Berikut penjelasan tipe-tipe pada bayangan:

###### Tipe bayangan

| [properti]          | Keterangan                                                 |
| ------------------- | ---------------------------------------------------------- |
| `shadow`            | Menerapkan bayangan normal                                 |
| `shadow-small`      | Menerapkan bayangan kecil                                  |
| `shadow-large`      | Menerapkan bayangan lebih besar                            |
| `shadow-dark`       | Menerapkan bayangan normal untuk latar belakang gelap      |
| `shadow-dark-small` | Menerapkan bayangan kecil untuk latar belakang gelap       |
| `shadow-dark-large` | Menerapkan bayangan lebih besar untuk latar belakang gelap |

<br />

### Penyesuaian Gaya

| Properti              | Keterangan                                      |
| --------------------- | ----------------------------------------------- |
| `--shadow`            | Bayangan normal                                 |
| `--shadow-small`      | Bayangan kecil                                  |
| `--shadow-large`      | Bayangan lebih besar                            |
| `--shadow-dark`       | Bayangan normal untuk latar belakang gelap      |
| `--shadow-dark-small` | Bayangan kecil untuk latar belakang gelap       |
| `--shadow-dark-large` | Bayangan lebih besar untuk latar belakang gelap |

<br />
