## Kartu

Komponen kartu merupakan sebuah komponen kontainer konten. Kartu juga menyediakan opsi penyesuaian warna latar belakang, warna teks, garis tepi, jarak konten dengan garis tepi, dan nilai tumpul sudut kartu.

###### Tipe kartu

| Class            | Keterangan             |
| ---------------- | ---------------------- |
| `card-primary`   | Warna kartu utama      |
| `card-secondary` | Warna kartu sekunder   |
| `card-success`   | Warna kartu berhasil   |
| `card-warning`   | Warna kartu peringatan |
| `card-danger`    | Warna kartu bahaya     |
| `card-info`      | Warna kartu info       |

<br />

### Penyesuaian Gaya

Komponen navigasi lokasi memiliki opsi penyesuaian gaya diantaranya:

| Properti        | Keterangan                                    |
| --------------- | --------------------------------------------- |
| `--bg-card`     | Mengatur warna latar belakang kartu           |
| `--text-card`   | Mengatur warna teks pada kartu                |
| `--space-card`  | Mengatur jarak konten dengan garis tepi kartu |
| `--border-card` | Mengatur garis tepi pada kartu                |
| `--corner-card` | Mengatur tumpul sudut kartu                   |

<br />

Penyesuaian gaya ini dapat diterapkan dengan menggunakan berbagai cara berikut:

##### # Melalui selektor `:root`

Pada file `.css` Anda, penyesesuaian melalui selektor global yaitu menggunakan `:root` dapat dilakukan dengan cara berikut:

<div className='alert-warning'>
	<span>
		Ingat bahwa penggunaan selektor <code>:root</code> dapat mempengaruhi
		gaya komponen kartu secara global/menyeluruh.
	</span>
</div>

##### # Melalui selektor elemen tertentu

Pada file `.css` Anda, menerapkan gaya komponen kartu untuk setiap elemen tertentu juga dapat dilakukan.
