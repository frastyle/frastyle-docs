Selain parameter warna diatas, terdapat parameter warna lain yang dapat digunakan yaitu `--primary` dan `--secondary`. Namun, kedua parameter ini belum memiliki standar nilai. Tentu dari parameter warna yang sudah tersedia, Anda dapat menggunakan dan menyesuaikan warnanya.

### Latar Belakang

Penerpan warna pada latar belakang elemen dapat dilakukan dengan menambahkan class `bg-` yang diikuti dengan parameter warna.

| Class            | Keterangan                      |
| ---------------- | ------------------------------- |
| `bg-primary`     | Warna latar belakang utama      |
| `bg-secondary`   | Warna latar belakang sekunder   |
| `bg-success`     | Warna latar belakang berhasil   |
| `bg-warning`     | Warna latar belakang peringatan |
| `bg-danger`      | Warna latar belakang bahaya     |
| `bg-info`        | Warna latar belakang info       |
| `bg-black`       | Warna latar belakang hitam      |
| `bg-dark`        | Warna latar belakang gelap      |
| `bg-dark-[1-5]`  | Warna latar belakang gelap 1-5  |
| `bg-white`       | Warna latar belakang putih      |
| `bg-light`       | Warna latar belakang cerah      |
| `bg-light-[1-5]` | Warna latar belakang cerah 1-5  |

<br />

##### # Contoh Penggunaan

<div className='bg-primary white text-center p-1'>
	<span>Latar belakang utama</span>
</div>
<div className='bg-dark-3 white text-center p-1'>
	<span>Latar belakang gelap-3</span>
</div>

<CodeBox lang='html' className='mt-1 mb-3'>{`
<div class='bg-primary'>
    <span>Latar belakang utama</span>
</div>
<div class='bg-dark-3'>
    <span>Latar belakang gelap-3</span>
</div>
`}</CodeBox>

### Teks

Warna dalam elemen teks dapat diterapkan hanya dengan menambahkan class parameter warna.

| Class         | Keterangan            |
| ------------- | --------------------- |
| `primary`     | Warna teks utama      |
| `secondary`   | Warna teks sekunder   |
| `success`     | Warna teks berhasil   |
| `warning`     | Warna teks peringatan |
| `danger`      | Warna teks bahaya     |
| `info`        | Warna teks info       |
| `black`       | Warna teks hitam      |
| `dark`        | Warna teks gelap      |
| `dark-[1-5]`  | Warna teks gelap 1-5  |
| `white`       | Warna teks putih      |
| `light`       | Warna teks cerah      |
| `light-[1-5]` | Warna teks cerah 1-5  |

<br />

##### # Contoh Penggunaan

<div className='primary text-center p-1'>
	<span>Teks utama</span>
</div>
<div className='dark-3 text-center p-1'>
	<span>Teks gelap-3</span>
</div>

<CodeBox lang='html' className='mt-1 mb-3'>{`
<div class='primary'>
    <span>Teks utama</span>
</div>
<div class='dark-3'>
    <span>Teks gelap-3</span>
</div>
`}</CodeBox>

### Garis Tepi

Penerapan warna untuk garis tepi sebuah elemen dapat diterapkan hanya dengan menambahkan class `outline-` yang diikuti dengan parameter warna.

| Class                 | Keterangan                  |
| --------------------- | --------------------------- |
| `outline-primary`     | Warna garis tepi utama      |
| `outline-secondary`   | Warna garis tepi sekunder   |
| `outline-success`     | Warna garis tepi berhasil   |
| `outline-warning`     | Warna garis tepi peringatan |
| `outline-danger`      | Warna garis tepi bahaya     |
| `outline-info`        | Warna garis tepi info       |
| `outline-black`       | Warna garis tepi hitam      |
| `outline-dark`        | Warna garis tepi gelap      |
| `outline-dark-[1-5]`  | Warna garis tepi gelap 1-5  |
| `outline-white`       | Warna garis tepi putih      |
| `outline-light`       | Warna garis tepi cerah      |
| `outline-light-[1-5]` | Warna garis tepi cerah 1-5  |

<br />

##### # Contoh Penggunaan

	<span>Garis tepi utama</span>
