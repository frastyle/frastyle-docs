Selain parameter warna diatas, terdapat parameter warna lain yang dapat digunakan yaitu `--primary` dan `--secondary`. Namun, kedua parameter ini belum memiliki standar nilai. Tentu dari parameter warna yang tersedia, Anda dapat menggunakan dan menyesuaikan warnanya.

### Latar Belakang

Penerpan warna pada latar belakang elemen dapat dilakukan dengan menambahkan class `bg-` yang diikuti dengan parameter warna.

| Class            | Keterangan                 |
| ---------------- | -------------------------- |
| `bg-primary`     | Latar belakang utama       |
| `bg-secondary`   | Latar belakang sekunder    |
| `bg-success`     | Latar belakang berhasil    |
| `bg-warning`     | Latar belakang peringatan  |
| `bg-danger`      | Latar belakang bahaya      |
| `bg-info`        | Latar belakang warna info  |
| `bg-black`       | Latar belakang warna hitam |
| `bg-dark`        | Latar belakang gelap       |
| `bg-dark-[1-5]`  | Latar belakang gelap 1-5   |
| `bg-white`       | Latar belakang warna putih |
| `bg-light`       | Latar belakang cerah       |
| `bg-light-[1-5]` | Latar belakang cerah 1-5   |

<br />

##### # Contoh Penggunaan

<div className='bg-primary white text-center p-1'>
	<span>Latar belakang utama</span>
</div>
<div className='bg-dark-3 white text-center p-1'>
	<span>Latar belakang gelap-3</span>
</div>

<CodeBox lang='html'>{`
<div class='bg-primary'>
    <span>Latar belakang utama</span>
</div>
<div class='bg-dark-3'>
    <span>Latar belakang gelap-3</span>
</div>
`}</CodeBox>

### Teks

Warna dalam elemen teks dapat diterapkan hanya dengan menambahkan class parameter warna.

| Class         | Keterangan       |
| ------------- | ---------------- |
| `primary`     | Teks utama       |
| `secondary`   | Teks sekunder    |
| `success`     | Teks berhasil    |
| `warning`     | Teks peringatan  |
| `danger`      | Teks bahaya      |
| `info`        | Teks warna info  |
| `black`       | Teks warna hitam |
| `dark`        | Teks gelap       |
| `dark-[1-5]`  | Teks gelap 1-5   |
| `white`       | Teks warna putih |
| `light`       | Teks cerah       |
| `light-[1-5]` | Teks cerah 1-5   |

<br />

##### # Contoh Penggunaan

<div className='primary text-center p-1'>
	<span>Teks utama</span>
</div>
<div className='dark-3 text-center p-1'>
	<span>Teks gelap-3</span>
</div>

<CodeBox lang='html'>{`
<div class='primary'>
    <span>Teks utama</span>
</div>
<div class='dark-3'>
    <span>Teks gelap-3</span>
</div>
`}</CodeBox>
