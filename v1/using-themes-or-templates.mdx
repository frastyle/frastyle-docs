## Tema & Template

### Pengenalan

Salah satu fitur dari Frastyle CSS Framework adalah menghadirkan adanya penyesuaian baik menggunakan tema, template, atau dengan membuat penyesuaian gaya Anda sendiri.

### Tema

Tim Frastyle Dev. dan komunitas telah menyediakan berbagai macam variasi style tema yang keren-keren. Tema ini dapat langsung Anda gunakan setelah melakukan instal Frastyle Framework. Seluruh daftar tema tersebut terdapat di dalam [Dunia Tema Frastyle](../tema).

> ###### Rekomendasi:
>
> Pilih metode instalasi tema sesuai dengan instalasi framework.
>
> Contoh: _Jika Anda melakukan instalasi framework menggunakan NPM/Yarn, kami merekomendasikan instalasi tema juga menggunakan NPM/Yarn._

<br />

Instalasi tema dapat dilakukan dengan menggunakan 3 metode instalasi yaitu:

##### # CDN (Content Delivery Network)

Masukkan perintah berikut ke file `.html` di dalam elemen tag `<head>`, dibawah tag Frastyle framework.

Contoh:

<CodeBox lang='html'>{`
<!-- Frastyle Framework -->
<!-- <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frastyle-css/dist/css/frastyle.min.css"> -->
<!-- Tema Frastyle -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@frastyle/simvle-theme/dist/css/simvle.min.css">
`}</CodeBox>

##### # Manejer Paket (Package Manager)

Anda dapat menggunakan Yarn/NPM untuk melakukan instalasi tema. Kunjungi [npmjs.com](https://www.npmjs.com/search?q=frastyle) atau [yarnpkg.com](https://yarnpkg.com/?q=frastyle) untuk mencari tema Frastyle dan melakukan instalasi.

Setelah selesai melakukan instalasi Yarn/NPM, masukkan tema dengan melakukan `@import` tema ke dalam file `.css` proyek Anda.

Contoh:

<CodeBox lang='css'>{`
/* @import "./node_modules/frastyle-css/dist/css/frastyle.min.css"; */
/* Tambahkan tema dibawah file framework */
@import "./node_modules/@frastyle/simvle-theme/dist/css/simvle.min.css";
`}</CodeBox>

##### # Unduh Tema

Template awal website besserta file framework dan tema yang sudah terkompresi telah dijadikan satu dalam file `.zip/.tar`. Cukup dengan mengunduh dan ekstrak file, proyek website beserta tema sudah siap digunakan.

### Template

Selain menyediakan tema, Tim Frastyle Dev. dan komunitas kami juga menyediakan template-template website mulai dari satu halaman website yang siap digunakan, struktur dasar website, atau komponen tertentu.

Kemudian bagaimana cara menggunakannya? Anda tinggal mengunjungi halaman [Koleksi Template Frastyle](../template), pilih dan unduh template. Kemudian ekstrak dan masukkan template ke dalam proyek Anda. Selesai.

### Penyesuaian Gaya Sendiri

Gaya tema kurang sesuai dengan yang Anda mau? Frastyle dapat dengan mudah menyesuaikan gaya tema sesuai dengan yang Anda inginkan. Bagaimana caranya? Coba ikuti panduan penyesuaian tiap elemen/komponen framework yang dijelaskan pada bagian **'Penyesuaian'** di setip elemen/komponen.
