## Margin & Padding

### Pengenalan

***Margin*** merupakan jarak sisi luar dari elemen atau dapat diartikan juga sebagai jarak antar elemen.

***Padding*** merupakan jarak sisi dalam dari elemen atau dapat diartikan juga sebagai jarak antara suatu elemen dengan isi elemen tersebut.

<img src={ MarginPadding } className="fit-contain" width="100%" height="96px" />

##### # Catatan

Class untuk menuliskan properti margin dan padding yang berlaku di semua breakpoint menggunakan format `[properti][sisi]-[ukuran]`, sedangkan untuk breakpoint yang spesifik menggunakan format `[properti][sisi]-[breakpoit]-[ukuran]`.

Ingin menggunakan untuk tata letak sistem grid Frastyle? Coba untuk mempertimbangkan penggunaan [class Gap](./grids-systems).

###### Penjelasan format

| [properti]            | Keterangan                 |
| :--------------: | -------------------------- |
| `m`     | Menerapkan ke dalam properti **'margin'**      |
| `p`   | Menerapkan ke dalam properti **'padding'**    |

<br />

| [sisi]            | Keterangan                 |
| :--------------: | -------------------------- |
| `x`     | Mengatur nilai sisi **kanan** dan **kiri**       |
| `y`   | Mengatur nilai sisi **atas** dan **bawah**    |
| `t`     | Mengatur nilai sisi **atas**      |
| `r`   | Mengatur nilai sisi **kanan**    |
| `b`     | Mengatur nilai sisi **bawah**      |
| `l`   | Mengatur nilai sisi **kiri**    |

<br />

Penggunaan `[sisi]` bersifat opsional sehingga apabila tidak menggunakan parameter ini, maka margin/padding akan diaplikasikan ke seluruh sisi elemen dengan nilai yang sama.

<br />

| [ukuran]            | Keterangan                 |
| :--------------: | -------------------------- |
| `0`     | Menerapkan jarak sebesar **0** atau untuk menghilangkan jarak     |
| `1`   | Menerapkan jarak sebesar **16px** (standar)   |
| `2`     | Menerapkan jarak sebesar **32px** (standar)      |
| `3`   | Menerapkan jarak sebesar **48px** (standar)    |
| `4`     | Menerapkan jarak sebesar **64px** (standar)      |
| `5`   | Menerapkan jarak sebesar **80px** (standar)   |
| `6`   | Menerapkan jarak sebesar **96px** (standar)    |
| `7`     | Menerapkan jarak sebesar **112px** (standar)      |
| `8`   | Menerapkan jarak sebesar **128px** (standar)    |
| `9`     | Menerapkan jarak sebesar **144px** (standar)      |
| `auto`   | Menerapkan jarak **otomatis** (hanya berlaku pada margin)    |

<br />

Penjelasan mengenai format **[breakpoint]** dapat dilihat pada menu [Tata Letak: Responsif](./responsive).

### Pemosisian ke Tengah

Melakukan pemosisian ke tengah secara horizontal untuk konten yang memiliki properti `display: block;` dan `width` dapat dilakukan dengan menambahkan class `m-auto` atau `mx-auto`.

<div class='m-auto' style={{width: 'fit-content'}}>
    <span class='bg-light-2'>Konten berada di tengah</span>
</div>

<CodeBox lang='html'>{`
<div class='m-auto' style="width: fit-content;">
    <span class='bg-light-2'>Konten berada di tengah</span>
</div>
`}</CodeBox>
