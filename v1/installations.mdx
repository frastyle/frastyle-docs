## Instalasi

Frastyle CSS Framework memiliki beberapa cara mudah untuk instalasi ke dalam proyek Anda. Cara yang paling sering digunakan adalah menginstal menggunakan CDN atau _Content Delivery Network_. Mengapa CDN? karena, mampu meningkatkan kecepatan website saat diakses. Cukup dengan menambahkan beberapa baris kode ke proyek untuk menginstal CDN, Frastyle siap untuk Anda gunakan.

Ingin menggunakan metode instalasi lain? Tenang, kami menyediakan metode lain untuk instalasi, yaitu menggunakan _package manager_ atau secara manual dengan mengunduh file template awal proyek.

> _Kami menyarankan menggunakan framework versi terakhir yang telah setabil untuk digunakan. Namun apabila ingin menggunakan versi sebelumnya, Anda dapat temukan di [Catatan Versi Rilis](https://github.com/frastyle/frastyle-css/blob/main/CHANGELOG.md)._

<br />

### CDN (Content Delivery Network)

Cukup dengan memasukkan perintah berikut ke file `.html` di dalam elemen tag `<head>`, Anda sudah melakukan instalasi Frastyle framework.

<CodeBox lang="xml" className='mb-3'>
{`<!-- Frastyle Framework -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frastyle-css/dist/css/frastyle.min.css">`}
</CodeBox>

### Manejer Paket (Package Manager)

Sering menggunakan package manager? Cukup dengan menjalankan salah satu kode berikut pada CLI proyek, Anda sudah bisa menggunakan Frastyle framework.

Anda menggunakan Yarn? pakai kode berikut:

<CodeBox lang='bash' className='mb-1'>{`yarn add frastyle`}</CodeBox>

atau Anda menggunakan NPM? pakai kode berikut:

<CodeBox lang='bash' className='mb-1'>{`npm install frastyle --save`}</CodeBox>

Setelah selesai melakukan instalasi Yarn/NPM, masukkan kode berikut ke dalam file `.css` proyek Anda untuk menggunakan framework.

<CodeBox
	lang='css'
	className='mb-3'>{`@import "./node_modules/frastyle-css/dist/css/frastyle.min.css";`}</CodeBox>

### Unduh Framework

Template awal website besserta file framework yang sudah terkompresi. Cukup dengan mengunduh dan ekstrak file, proyek website sudah siap digunakan.

<a

	target='_blank'
	className='box-s-5 box-l-3 button-primary mx-0'>
	Unduh
</a>

<br />

Selain menggunakan `frastyle.min.css` Anda juga dapat menggunakan file framework dari daftar berikut:

```
📂css/
├─  frastyle.css
├─  frastyle.min.css
├─  frastyle-base.css
├─  frastyle-base.min.css
├─  frastyle-element.css
├─  frastyle-element.min.css
├─  frastyle-grid.css
└─  frastyle-grid.min.css
```
